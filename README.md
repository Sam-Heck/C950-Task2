# C950 - Task 2

- Processes CSV package data
- Calculates optimal routes for package delivery given constraints
- Outputs a summary of packages at user-specified time intervals.

## How to Run
No external dependencies are required (built using Python 3)

1. Run the main script from the root directory:

    ```bash
    python src/main.py
    ```

2. Enter a time when prompted in the following format: `HH:MM:SS` (e.g., 09:30:00)
